# FinAgentAI – Multi-Agent Financial Intelligence & Stock Recommendation Platform

Tech Stack: Python, Streamlit, Machine Learning, XGBoost, Yahoo Finance API, NLP, Plotly, Scikit-Learn

Features

Multi-agent architecture for market, sentiment, risk, and portfolio analysis.
Technical indicators including RSI and moving averages.
Financial news sentiment analysis.
ML-based stock price forecasting.
Interactive Streamlit dashboard for investment insights.
